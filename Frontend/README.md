# ⌚ WatchVault – Vendor & Customer Frontend

## 📌 Project Title
WatchVault – Luxury Watch E-commerce Platform (Frontend)

---

## 📖 Project Description
WatchVault is a modern e-commerce platform built for buying and selling luxury watches. 

The frontend provides:
- Customer browsing experience
- Wishlist & cart functionality
- Checkout flow
- Vendor dashboard for managing products
- Orders and analytics view

This frontend connects to a Node.js + Supabase backend.

---

## 🚀 Features

### 👤 Customer Features
- Browse watches catalog
- Filter by brand and price
- View product details
- Add to cart
- Wishlist functionality
- Checkout and order history

### 🏪 Vendor Features
- Add new products with image upload
- Manage products (edit/delete)
- View orders
- Analytics dashboard
- Secure vendor routes

---

## 🛠 Tech Stack Used

- React (Vite)
- React Router
- Tailwind CSS
- Axios
- Firebase Authentication
- Supabase (via backend API)
